# Scikit-Learn Machine Learning Projects (Python + Scikit-Learn)

## Project Description
This repository contains a collection of machine learning projects and experiments built using Python and the Scikit-learn library.

The scripts demonstrate fundamental machine learning concepts such as training models, classification, prediction, and evaluating model performance using real datasets.

These projects were created to practice machine learning techniques and understand how algorithms work using the Scikit-learn framework.

---

## Technologies
- Python
- Scikit-learn
- NumPy
- Matplotlib

---

## Projects Included

### Handwritten Digit Recognition (K-Nearest Neighbors)
File: sklearn_digits_KNeighborsClassifier.py

This project trains a machine learning model to recognize handwritten digits using the Scikit-learn digits dataset.

The program:
- loads the digits dataset
- splits the dataset into training and testing data
- trains a **K-Nearest Neighbors classifier**
- predicts digit labels
- evaluates model accuracy
- visualizes example digits

This demonstrates basic **classification using the KNN algorithm**.

## How to Run

Install the required libraries:

pip install scikit-learn  
pip install numpy  
pip install matplotlib  

Run any script:

python filename.py

Example:

python sklearn_digits_KNeighborsClassifier.py

The program trains a machine learning model to classify handwritten digits and prints the model accuracy.
